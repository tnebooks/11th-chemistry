**ஒரு கலனில் சம எண்ணிக்கையுள்ள ஹைட்ரஜன் மற்றும் ஆக்சிஜன் மோல்கள் ஒரு சிறு துளை வழியே வெளியேறுகின்றன. பாதி ஹைட்ரஜன் வெளியேற தேவைப்படும் அதே நேரத்தில் வெளியேறும் ஆக்சிஜனின் பின்ன அளவு (NEET Phase I)**

```markdown
கிரகாம் விதி: r ∝ 1/√M.
r(O₂)/r(H₂) = √(M_H₂/M_O₂) = √(2/32) = √(1/16) = 1/4.
1/2 மோல் H₂ வெளியேற தேவைப்படும் நேரத்தில், O₂ வெளியேறியது = (1/4) × (1/2) = 1/8 மோல்.
O₂ வெளியேறிய பின்னம் = (1/8)/1 = 1/8.
சரியான விடை (இ) 1/8.
```
