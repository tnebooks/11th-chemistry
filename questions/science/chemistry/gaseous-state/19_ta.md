**P, Q, R மற்றும் S என்ற நான்கு வாயுக்களின் 'b' மதிப்பு சமம் ஆனால் 'a' மதிப்பு Q < R < S < P (a, b வாண்டர் வால்ஸ் மாறிலிகள்). குறிப்பிட்ட வெப்பநிலையில் நான்கு வாயுக்களில் மிக எளிதாக ஆவியாகும் வாயு**

```markdown
வாண்டர் வால்ஸ் மாறிலி 'a' என்பது இடைமூலக்கூறு கவர்ச்சி விசைகளின் அளவுகோலாகும். 'a' மதிப்பு அதிகமாக இருந்தால், இடைமூலக்கூறு விசைகள் வலிமையானவை, மேலும் திரவமாக்கல் எளிதானது. P அதிக 'a' மதிப்பைக் கொண்டுள்ளது, எனவே P மிக எளிதாக திரவமாக்கப்படும்.
சரியான விடை (அ) P.
```
