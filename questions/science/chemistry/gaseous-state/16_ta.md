**ஒரு சமவெப்ப அழுத்த நிலையில் CₙH₂ₙ₋₂ என்ற மூலக்கூறு வாய்ப்பாடு கொண்ட ஹைட்ரோகார்பனை விட ஹைட்ரஜன் வாயு 3√3 மடங்கு விரவுதல் வீதம் கொண்டது எனில் 'n' மதிப்பு என்ன?**

```markdown
r(H₂)/r(ஹைட்ரோகார்பன்) = √(M_ஹைட்ரோகார்பன்/M_H₂) = 3√3
இருபுறமும் வர்க்கப்படுத்த: M_ஹைட்ரோகார்பன்/M_H₂ = 27
M_ஹைட்ரோகார்பன் = 27 × 2 = 54
CₙH₂ₙ₋₂ க்கு: 12n + 1(2n-2) = 54 → 14n - 2 = 54 → 14n = 56 → n = 4.
சரியான விடை (ஆ) 4.
```
