**-I effect is shown by**

```markdown
-I effect (electron withdrawing) is shown by halogens like -Cl, -Br, -F, and other electronegative groups. -CH₃ shows +I effect (electron donating).
Both -Cl and -Br show -I effect.
Correct answer is (c) both (a) and (b).
```
