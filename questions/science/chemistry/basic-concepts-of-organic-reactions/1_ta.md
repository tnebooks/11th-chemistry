  - "(A) நீக்க வினை, (B) பதிலீட்டு வினை மற்றும் (C) சேர்க்கை வினை"

**(A) CH₃CH₂CH₂Br + KOH → CH₃-CH=CH₂ + KBr + H₂O; (B) (CH₃)₃CBr + KOH → (CH₃)₃COH + KBr; (C) Br₂ → Br-Br. மேற்கண்டுள்ள வினைகளுக்கு, பின்வரும் எந்த கூற்று சரியானது?**

```markdown
(A) ஆல்கைல் ஹலைடு KOH உடன் வினைப்பட்டு ஆல்க்கீன் தருகிறது → நீக்க வினை (டிஹைட்ரோஹாலஜனேற்றம்)
(B) மூன்றாம் நிலை ஆல்கைல் ஹலைடு KOH உடன் வினைப்பட்டு ஆல்கஹால் தருகிறது → பதிலீட்டு வினை (SN¹)
(C) Br₂ ஆல்க்கீனுடன் சேர்க்கப்பட்டு டைபுரோமைடு தருகிறது → சேர்க்கை வினை
சரியான விடை (ஈ).
```
