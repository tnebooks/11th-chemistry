**பின்வரும் வாய்ப்புகளில், கொடுக்கப்பட்ட வரிசைகளுக்கு அவற்றிற்கு எதிராக குறிப்பிடப்பட்டுள்ள பண்பினைப் பார்த்து சரியாக அமைந்திருக்காத வரிசை இடம்பெற்றுள்ள வாய்ப்பு எது? (NEET 2016 Phase I)**

```markdown
F-ன் எலக்ட்ரான் நாட்டம் Cl-ஐ விட குறைவு (F = -328, Cl = -349 kJ/mol). சரியான வரிசை: I < Br < F < Cl. விருப்பம் (அ) I < Br < Cl < F என்று தவறாகக் கூறுகிறது. மற்ற விருப்பங்கள் அனைத்தும் சரி.
சரியான விடை (அ).
```
