**Which one of the following arrangements represent the correct order of least negative to most negative electron gain enthalpy?**

```markdown
Electron gain enthalpy (negative values): Ca (least negative) < Al < C < O < F (most negative). Ca has the least tendency to gain electron, F has the highest. Correct order: Ca < Al < C < O < F.
Correct answer is (d) Ca < Al < C < O < F.
```
