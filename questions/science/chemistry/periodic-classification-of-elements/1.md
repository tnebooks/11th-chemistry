**What would be the IUPAC name for an element with atomic number 222?**

```markdown
According to IUPAC nomenclature: 200 = bi, 20 = bi, 2 = bi → bibibium. The roots are: 0=nil, 1=un, 2=bi, 3=tri, 4=quad, 5=pent, 6=hex, 7=sept, 8=oct, 9=enn. 222 → bi-bi-bium = bibibium.
Correct answer is (d) bibibium.
```
