**பெர்ரஸ் ஆக்சலேட்டின் சமான நிறை**

```markdown
FeC₂O₄ → Fe³⁺ + 2CO₂ + 3e⁻
Fe²⁺ → Fe³⁺ + 1e⁻ (1 எலக்ட்ரான்)
C₂O₄²⁻ → 2CO₂ + 2e⁻ (2 எலக்ட்ரான்கள்)
மொத்தம் 3 எலக்ட்ரான்கள்.
சமான நிறை = மோலார் நிறை / 3
சரியான விடை (இ) பெர்ரஸ் ஆக்சலேட்டின் மோலார் நிறை / 3.
```
