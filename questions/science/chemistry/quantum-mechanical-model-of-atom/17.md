**Which of the following sets of quantum number is not possible? (i) n=3, l=0, m=0, s=+1/2; (ii) n=2, l=2, m=1, s=-1/2; (iii) n=4, l=3, m=-2, s=+1/2; (iv) n=1, l=0, m=-1, s=+1/2; (v) n=3, l=4, m=3, s=-1/2**

```markdown
(ii) Not possible - l=2, for n=2, l can be 0 or 1 only (l < n).
(iv) Not possible - m = -1, for l=0, m can only be 0.
(v) Not possible - l=4, for n=3, l < n (l = 0,1,2 only).
Correct answer is (b) (ii), (iv) and (v).
```
