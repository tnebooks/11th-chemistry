**What is the maximum numbers of electrons that can be associated with the following set of quantum numbers? n = 3, l = 1 and m = -1**

```markdown
n = 3, l = 1 represents 3p orbital. m = -1 represents one specific p orbital. A single orbital can hold a maximum of 2 electrons (s = +1/2 and -1/2).
Correct answer is (c) 2.
```
