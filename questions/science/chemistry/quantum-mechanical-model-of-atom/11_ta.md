**ஒரு துணைக்கூட்டில் உள்ள அதிகபட்ச எலக்ட்ரான்களின் எண்ணிக்கையினைக் குறிப்பிடுவது**

```markdown
ஒரு துணைக்கூட்டில் (subshell) அதிகபட்சமாக 2(2l + 1) = 4l + 2 எலக்ட்ரான்கள் இருக்க முடியும். l = 0 (s) → 2, l = 1 (p) → 6, l = 2 (d) → 10, l = 3 (f) → 14.
2n² என்பது ஒரு கூட்டில் (shell) உள்ள அதிகபட்ச எலக்ட்ரான்களின் எண்ணிக்கை.
சரியான விடை (இ) 4l + 2.
```
