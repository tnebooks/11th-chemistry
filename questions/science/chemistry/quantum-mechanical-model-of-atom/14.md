**Assertion: Number of radial and angular nodes for 3p orbital are 1, 1 respectively. Reason: Number of radial and angular nodes depends only on principal quantum number.**

```markdown
For 3p: Radial nodes = n - l - 1 = 3 - 1 - 1 = 1; Angular nodes = l = 1. Assertion is true. Reason is false - radial nodes depend on both n and l, angular nodes depend on l only.
Correct answer is (c) assertion is true but reason is false.
```
