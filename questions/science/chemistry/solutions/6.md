**Which one of the following is incorrect for ideal solution?**

```markdown
For ideal solution: ΔH_mix = 0, ΔU_mix = 0, ΔV_mix = 0. But ΔG_mix = ΔH_mix - TΔS_mix = -TΔS_mix ≠ 0 (since ΔS_mix > 0). So ΔG_mix = 0 is incorrect.
Correct answer is (d) ΔG_mix = 0.
```
