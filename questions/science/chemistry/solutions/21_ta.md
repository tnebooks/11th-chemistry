**ஒரு கரைசலின், செறிவிற்கு (C mol L⁻¹) எதிரான சவ்வூடு பரவல் அழுத்தம் (π) வரைபடம் நேர்க்கோடு தருகிறது. இதன் சாய்வு 310R. 'R' என்பது வாயு மாறிலி. சவ்வூடு பரவல் அழுத்தம் அளவிடப்பட்ட வெப்பநிலை**

```markdown
π = CRT → π vs C வரைபடத்தின் சாய்வு = RT
சாய்வு = RT = 310R → T = 310 K = 37°C
சரியான விடை (இ) 37°C.
```
