**நல்லியல் கரைசலுக்கு பின்வருவனவற்றுள் எது தவறானது?**

```markdown
நல்லியல் கரைசலுக்கு: ΔH_mix = 0, ΔU_mix = 0, ΔV_mix = 0. ஆனால் ΔG_mix = ΔH_mix - TΔS_mix = -TΔS_mix ≠ 0 (ஏனெனில் ΔS_mix > 0). எனவே ΔG_mix = 0 என்பது தவறானது.
சரியான விடை (ஈ) ΔG_mix = 0.
```
