**In an adiabatic process, which of the following is true?**

```markdown
In an adiabatic process, there is no exchange of heat between the system and the surroundings. Therefore, q = 0. From the first law: ΔU = q + w = w (since q = 0). PΔV = 0 is true for isochoric process.
Correct answer is (b) q = 0.
```
