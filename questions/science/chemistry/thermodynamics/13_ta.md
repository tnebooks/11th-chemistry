answers:
  - "ΔH < ΔU"

**ஒரு மூடிய கலனில், ஒரு மோல் அம்மோனியா மற்றும் ஒரு மோல் ஹைட்ரஜன் குளோரைடு கலக்கப்பட்டு அம்மோனியம் குளோரைடு வாயு உருவாகிறது எனில்**

```markdown
வினை: NH₃(g) + HCl(g) → NH₄Cl(s)
Δn_g = வாயு விளைபொருட்களின் மோல்கள் - வாயு வினைபடுபொருட்களின் மோல்கள் = 0 - 2 = -2
ΔH = ΔU + Δn_g RT = ΔU - 2RT
எனவே, ΔH < ΔU.
சரியான விடை (ஈ) ΔH < ΔU.
```
