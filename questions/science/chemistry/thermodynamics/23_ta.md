**ஒரு திரவத்தின் மோலார் ஆவியாதல் வெப்பம் 4.8 kJ mol⁻¹. அதன் எந்திரோபி மாற்ற மதிப்பு 16 J K⁻¹ mol⁻¹ எனில் அந்த திரவத்தின் கொதிநிலை**

```markdown
கொதிநிலையில்: ΔG = 0, எனவே ΔH = TΔS
T = ΔH/ΔS = 4.8 × 10³ / 16 = 4800/16 = 300 K = 27°C
சரியான விடை (ஆ) 27°C.
```
