**Al₂O₃ மற்றும் Cr₂O₃ ஆகியவற்றின் உருவாக்க என்தால்பி மதிப்புகள் முறையே -1596 kJ மற்றும் -1134 kJ. 2Al + Cr₂O₃ → 2Cr + Al₂O₃ என்ற வினைக்கு ΔH மதிப்பு**

```markdown
ΔH_reaction = ΣΔH_f(விளைபொருட்கள்) - ΣΔH_f(வினைபடுபொருட்கள்)
= [ΔH_f(Al₂O₃) + 2ΔH_f(Cr)] - [2ΔH_f(Al) + ΔH_f(Cr₂O₃)]
= (-1596 + 0) - (0 + (-1134)) = -1596 + 1134 = -462 kJ
சரியான விடை (ஈ) -462 kJ.
```
