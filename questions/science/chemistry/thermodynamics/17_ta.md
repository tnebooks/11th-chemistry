---
choices:
  - "(b + 2a)/2"
  - "(2a - b)/2"
  - "இவற்றில் ஏதுமில்லை"
answers:
  - "(b - 2a)/2"
---

**C(g) + O₂(g) → CO₂(g) ΔH = -a kJ; 2CO(g) + O₂(g) → 2CO₂(g) ΔH = -b kJ; எனில் C(g) + ½O₂(g) → CO(g) என்ற வினைக்கு ΔH மதிப்பு**

```markdown
சமன்பாடு (1): C + O₂ → CO₂, ΔH₁ = -a
சமன்பாடு (2): 2CO + O₂ → 2CO₂, ΔH₂ = -b
தேவைப்படும்: C + ½O₂ → CO
(1) - ½(2): C + O₂ - CO - ½O₂ → CO₂ - CO₂
C + ½O₂ - CO → 0
C + ½O₂ → CO
ΔH = ΔH₁ - ½ΔH₂ = -a - ½(-b) = -a + b/2 = (b - 2a)/2
சரியான விடை (இ) (b - 2a)/2.
```
