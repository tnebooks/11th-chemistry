**வெப்பம் மாறா செயல்முறையில் பின்வருவனவற்றுள் எது உண்மை?**

```markdown
வெப்பம் மாறா (அடியாபேடிக்) செயல்முறையில், அமைப்புக்கும் சுற்றுப்புறத்திற்கும் இடையே வெப்பப் பரிமாற்றம் இல்லை. எனவே, q = 0. முதல் விதியிலிருந்து: ΔU = q + w = w (q = 0 என்பதால்). PΔV = 0 என்பது மாறிலி கனஅளவு செயல்முறைக்கு உண்மை.
சரியான விடை (ஆ) q = 0.
```
