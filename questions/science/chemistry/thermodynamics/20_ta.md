**அனைத்து வெப்பநிலைகளிலும், ஒரு தன்னிச்சையான வினைக்கு சரியான வெப்ப இயக்கவியல் நிபந்தனைகள் (NEET Phase - I)**

```markdown
ஒரு வினை அனைத்து வெப்பநிலைகளிலும் தன்னிச்சையாக நிகழ: ΔG = ΔH - TΔS < 0.
ΔH < 0 (வெப்ப வெளியேற்றம்) மற்றும் ΔS > 0 (எந்திரோபி அதிகரிப்பு) எனில், அனைத்து வெப்பநிலைகளிலும் ΔG < 0 ஆகும். இது அனைத்து வெப்பநிலைகளிலும் தன்னிச்சையான தன்மைக்கான நிபந்தனையாகும்.
சரியான விடை (அ) ΔH < 0 மற்றும் ΔS > 0.
```
