**0°C மற்றும் 1 atm அழுத்தத்தில் 15.68 L மீத்தேன் மற்றும் புரோப்பேன் கலந்த வாயுக்கலவையை முழுமையாக எரிக்க, அதே வெப்ப அழுத்த நிலையில் 32 L ஆக்சிஜன் தேவைப்படுகிறது. இந்த எரிதல் வினையில் வெளியாகும் வெப்பத்தின் அளவு (kJ) (ΔHc(CH₄) = -890 kJ mol⁻¹ மற்றும் ΔHc(C₃H₈) = -2220 kJ mol⁻¹)**

```markdown
STP-ல், 1 மோல் வாயு = 22.4 L.
மொத்த மோல்கள் = 15.68/22.4 = 0.7 mol.
x = CH₄ மோல்கள், y = C₃H₈ மோல்கள் என்க.
x + y = 0.7
O₂ தேவை: CH₄ + 2O₂ → CO₂ + 2H₂O (2x)
C₃H₈ + 5O₂ → 3CO₂ + 4H₂O (5y)
2x + 5y = 32/22.4 = 1.4286
தீர்க்க: x = 0.5, y = 0.2
வெளியாகும் வெப்பம் = 0.5(-890) + 0.2(-2220) = -445 - 444 = -889 kJ
சரியான விடை (அ) -889 kJ.
```
