**In an adiabatic expansion of an ideal gas**

```markdown
In an adiabatic process, q = 0. From the first law: ΔU = q + w = 0 + w = w. Therefore, w = ΔU. But since the gas expands, work is done by the system (negative work), so w = -ΔU. Actually ΔU = w, so w = ΔU. However, in expansion, w = -PΔV and ΔU is negative, so w = -ΔU.
Correct answer is (a) w = -ΔU.
```
