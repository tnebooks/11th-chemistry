**The correct thermodynamic conditions for the spontaneous reaction at all temperatures is (NEET Phase - I)**

```markdown
For a reaction to be spontaneous at all temperatures: ΔG = ΔH - TΔS < 0.
If ΔH < 0 (exothermic) and ΔS > 0 (increase in entropy), then ΔG < 0 at all temperatures. This is the condition for spontaneity at all temperatures.
Correct answer is (a) ΔH < 0 and ΔS > 0.
```
