**PCl₅ ⇌ PCl₃ + Cl₂ என்ற வினையில், 0.5 மோல் PCl₅ உடன் தொடங்கும் போது x என்பது பிரிகையடைந்த பின்னமாகும். சமநிலையில் மொத்த மோல்களின் எண்ணிக்கை**

```markdown
PCl₅ ஆரம்பம் = 0.5 மோல். x மோல் பிரிகையடைகிறது.
சமநிலையில்: PCl₅ = 0.5 - x, PCl₃ = x, Cl₂ = x
மொத்த மோல்கள் = (0.5 - x) + x + x = 0.5 + x
சரியான விடை (ஆ) x + 0.5.
```
