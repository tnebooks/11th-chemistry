**In which of the following equilibrium, Kp and Kc are not equal?**

```markdown
Kp = Kc(RT)^Δn. When Δn = 0, Kp = Kc.
For options (a), (b), and (c), Δn = 0 (equal number of gaseous moles on both sides).
For option (d): PCl₅(g) ⇌ PCl₃(g) + Cl₂(g), Δn = 2 - 1 = 1. So Kp ≠ Kc.
Correct answer is (d).
```
