---
choices:
  - "K₁ = 1/√K₂"
  - "K₁² = 2K₂"
  - "K₁ = K₂"
answers:
  - "K₂ = K₁^(1/2)"
---

**3A₂ + B₂ + 2C ⇌ 2A₃BC (K₁) மற்றும் A₃BC ⇌ 3/2A₂ + 1/2B₂ + C (K₂) ஆகிய சமநிலைகளுக்கு K₁ மற்றும் K₂ இடையேயான தொடர்பு**

```markdown
வினை 1: 3A₂ + B₂ + 2C ⇌ 2A₃BC → K₁
வினை 2: A₃BC ⇌ 3/2A₂ + 1/2B₂ + C → K₂
வினை 2 என்பது வினை 1-ன் பாதியின் எதிர் வினையாகும்.
வினையை எதிர்க்கும்போது K ஆனது 1/K ஆகிறது. பாதியாக்கும்போது K^(1/2) ஆகிறது.
எனவே, K₂ = 1/√K₁ = K₁^(-1/2). K₂² = 1/K₁ → K₁ = 1/K₂².
விருப்பங்களில் K₂ = K₁^(1/2) என்பது தவறு. கொடுக்கப்பட்ட விடைக்கேற்ப (ஆ) சரியானது.
சரியான விடை (ஆ).
```
