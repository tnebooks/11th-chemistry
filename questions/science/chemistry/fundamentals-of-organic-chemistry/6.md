**Which one of the following names does not fit a real name?**

```markdown
3-Methyl-3-hexanone: In a ketone, the carbonyl carbon should have the lowest possible number. For a 6-carbon chain with methyl at C-3 and carbonyl at C-3, the parent chain should be chosen differently. The correct name should be 3-methyl-2-hexanone or similar. "3-Methyl-3-hexanone" is not a valid IUPAC name because the carbonyl carbon should get the lowest number.
Correct answer is (a).
```
