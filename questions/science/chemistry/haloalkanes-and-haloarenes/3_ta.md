answers:
  - "D < C < B < A"

**A) CCl₄, B) CHCl₃, C) CH₂Cl₂, D) CH₃Cl ஆகிய சேர்மங்களை அவற்றின் அடர்த்தியின் ஏறு வரிசையில் அமைக்க**

```markdown
அடர்த்தி குளோரின் அணுக்களின் எண்ணிக்கை அதிகரிக்க அதிகரிக்கிறது (அதிக மூலக்கூறு நிறை). CH₃Cl < CH₂Cl₂ < CHCl₃ < CCl₄. எனவே D < C < B < A.
சரியான விடை (அ) D < C < B < A.
```
