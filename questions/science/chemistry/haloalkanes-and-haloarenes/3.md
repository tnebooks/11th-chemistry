answers:
  - "D < C < B < A"
---

**Arrange the following compounds in increasing order of their density: A) CCl₄, B) CHCl₃, C) CH₂Cl₂, D) CH₃Cl**

```markdown
Density increases with increase in number of chlorine atoms (higher molecular weight). CH₃Cl < CH₂Cl₂ < CHCl₃ < CCl₄. So D < C < B < A.
Correct answer is (a) D < C < B < A.
```
